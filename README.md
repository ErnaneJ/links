## 💡 Use
- Clone este repositorio
  ```bash
  git clone https://github.com/ErnaneJ/me.git
  ```
- Instale as dependencias...
  
  ```bash
  cd me
  npm install
  ```

- Execute a aplicacao:

  ```bash
  npm run dev
  ```

Acesse [localhost:5000](http://localhost:5000).
